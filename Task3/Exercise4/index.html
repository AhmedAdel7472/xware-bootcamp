Section 1: Getting Started

Prerequisites

Before you begin, ensure you have the following installed:

  • Web browser (Chrome, Firefox, Safari, etc.)

Section 2: Features

  • Add Tasks: Enter a task description and click "Add" to add it to the list.
  • Delete Tasks: Click on the delete icon next to a task to remove it from the list.
  • Mark as Done: Click on a task to mark it as completed.

Section 3: Installation

Clone the repository from GitHub:

git clone https://github.com/your-username/simple-todo-list.git

Navigate to the project directory:

cd simple-todo-list

Section 4: Usage

  1. Open index.html in your web browser.
  2. Enter a task in the input field and click "Add" or press Enter to add it to your list.
  3. To delete a task, click on the delete icon.
  4. Click on a task to mark it as completed.

Section 5: Contributing

  1. Fork the repository.
  2. Create a new branch git checkout -b feature/improvement
  3. Make your changes.
  4. Commit your changes git commit -am 'Add some feature'
